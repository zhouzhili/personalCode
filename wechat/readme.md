WeChatWeather文件夹里的是天气查询小程序文件，该小程序只包含两个简单的页面。
	1. 检索页面，通过使用微信的定位功能，结合高德地图API获取当前城市信息，也提供手动填写城市信息，点击查询按钮进行查询。

	2. 信息展示页面，通过万年历的接口，根据给定的城市名称查询当前天气，并展示出来，如果未查询到则给出错误信息页面。
